## 기능 ⭐️

- About me 아래 숫자 클릭 시 증명서 사진
- Work Experience 활동 옆 사진 아이콘 클릭 시 갤러리 등장
- Project 링크 클릭 시 외부 링크(노션)로 이동
- Side Project 또한 블로그 글로 이동 가능
- 심심하지 않을 다양한 애니메이션
- contact 연락처 클릭시 바로 전화/메일 등 연결
